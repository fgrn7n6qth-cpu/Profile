## Projects

### Room Planner

공간 배치와 인테리어 구성을 도와주는 원룸 배치 웹 서비스입니다.  
사용자가 원룸 크기를 선택하고, 가구를 배치한 뒤 2D/3D 화면에서 결과를 확인할 수 있도록 구현했습니다.

#### 주요 구현 포인트

- 원룸 프로젝트 생성 및 최근 프로젝트 관리 기능 구현
- 가구 배치, 선택, 이동, 회전, 크기 조절이 가능한 2D 편집 화면 구현
- 같은 배치 데이터를 기반으로 3D 미리보기 화면 구현
- 배치 상태를 분석해 부족한 가구를 추천하는 기능 구현
- 추천 가구를 외부 상품 검색으로 연결하는 흐름 구현

#### 사용 기술

- Backend: Java, Spring Boot, REST API
- Frontend: React, HTML, CSS, JavaScript
- Database: MySQL
